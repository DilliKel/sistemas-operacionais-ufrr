## Threads e Fatorial

Seminário sobre multithreading com `pthread`, usando o cálculo do fatorial de números grandes (representado como vetor de dígitos, por estourar os tipos numéricos nativos) para comparar uma implementação sequencial com uma paralela.

Feito em dupla com **Natália Almada**. Detalhes completos, código-fonte, capturas e slides em [`seminario-threads-fatorial/`](./seminario-threads-fatorial).
